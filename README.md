# Stitch to Figma Assets

this repository is utilized to maintain all of the html content generated through google stitch (a ui/ux generator), we utilize stitch for our ui related work and then we import them to figma as required, and afterwards update, the generated html content can be highly valuable for developers when developing the figma page provided, in turn we are maintaing that intermediatery content of html within this repository for developers future reference, non of it is functional, purely for the sake of the code.

deployed url: https://stitch-to-figma-assets.pages.dev/
